## Install

Prerequisites: truffle installed

```
$ npm i -g @etherisc/gif-tools
$ git clone https://github.com/etherisc/GIF.git
$ cd GIF/gif-contracts
$ gif-tools select-resources
$ truffle compile --all
$ cp .env.sample .env
$ nano .env
$ truffle migrate --reset
```
